# Legacy Runtime

这里放的是旧实验台。

它不是当前推荐入口，但也不是准备立刻丢掉的废页。更准确一点说，它像一间还没完全搬空的旧工作室：里面有些东西已经显得太重、太杂，可也确实还留着一些以后可能要拆回来的能力。

这一层目前包含：

- `legacy/index.html`
  原来的大一体化实验页。
- `scripts/`
  只给旧实验台使用的实验脚本。

现在的默认策略是：

- 真正要继续长的主路，走 `index.html`
- 旧实验台主要用来回看历史流程，或者继续拆模块

所以，如果你只是想正常使用这套网页，先从新的入口开始；如果你是来考古、比对旧功能、搬回旧能力，再来这里。这样两边都不会互相绊脚。
